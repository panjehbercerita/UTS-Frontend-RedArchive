# [RED ARCHIVE] - Database Literatur Terlarang
### Tugas UTS Pemrograman Front-End

**Deskripsi Proyek:**
Aplikasi pendataan buku sederhana dengan tema "Noir / Bawah Tanah".
Aplikasi ini memungkinkan pengguna untuk menambahkan data buku, mencari buku, dan menghapus data.
Data tersimpan secara lokal di browser (LocalStorage) sehingga tidak hilang saat di-refresh.

**Fitur:**
* Input Data Buku (Judul, Penulis, Status, Sinopsis).
* Pencarian Data (Search Bar).
* Hapus Data.
* Penyimpanan Data Persisten (LocalStorage).
* Dark Mode / Noir Theme.
